# 诗网 Crossword

  

## 项目简介

  

诗网是一款基于HTML5 Canvas的创意文字游戏，玩家通过在网格上连接汉字来创作诗句。游戏从一个初始诗句开始，玩家可以选择现有诗句中的任意汉字作为连接点，向水平或垂直方向延伸创作新的诗句，形成不断扩展的诗句网络。

  

**v1.0.0 - 2026年2月24日发布**



  

## 核心玩法

- 输入一句喜欢的诗词**起始诗词** 字数在2-20个以内

**【编织】过程**：

1. 点击棋盘上任意一个已有汉字 

2. 选择**锚点**作为新诗与旧诗的交叉点

3. 点击同行/同列的**空格子** → 以该格为新诗起始点，方向朝锚字延伸

  - 如果**起始点**与**锚点**是**同一个字** 将出现 ←→↑↓ 四个箭头，选方向后从该字本身开始延伸

## 功能特性

### 📱 移动端优化

- **虚拟按钮**：左侧和右侧虚拟方向控制按钮

- **响应式布局**：自动适配手机和平板设备

## 游戏规则

  

### 基本规则

1. **开始游戏**：输入一句诗词开始游戏（至少2个字）

2. **选择连接点**：点击已有诗句中的任意汉字

3. **选择方向**：二次点击汉字显示方向箭头（↑↓←→）

4. **输入新字**：点击目标方向格子，输入新汉字

5. **确认连接**：诗句必须与已有诗句交叉重合，且包含新的汉字

6. **继续创作**：不断扩展诗句网络

  

### 操作说明

- **选择连接点**：点击已有汉字选择起点

- **选择方向**：再次点击汉字显示方向箭头

- **输入字符**：点击目标格子，输入新汉字

- **移动光标**：使用方向键移动输入位置

- **删除字符**：Backspace删除当前字符



### 快捷键

- **确认提交**：Enter提交诗句

- **取消编辑**：Esc退出编辑模式

- **撤销**：Ctrl+Z撤销上一步

- **重做**：Ctrl+Y或Ctrl+Shift+Z重做

  

### 验证规则

- 诗句中间不能有空格

- 新输入的字必须与已有字重合（冲突检测）

- 新诗句必须包含至少一个新字

- 不能录入已存在的诗句（去重检测）

## 开发指南

  

### 代码架构

项目采用模块化设计，主要分为以下几个部分：

  

1. **常量定义** - CELL_SIZE、INITIAL_POEM等

2. **游戏状态** - gameState对象管理所有状态

3. **输入管理** - InputManager统一处理输入

4. **触摸UI** - TouchUI提供移动端交互

5. **画布渲染** - draw系列函数负责视觉输出

6. **事件处理** - handle系列函数处理用户交互

7. **游戏逻辑** - 核心玩法实现

  

### 关键函数

  

#### 画布相关

- `drawCanvas()` - 主绘制函数，按顺序绘制所有元素

- `drawGrid()` - 绘制网格背景

- `drawPoems()` - 绘制所有诗句

- `drawHighlightedCells()` - 高亮选中的行列

- `drawInputCells()` - 绘制输入方格

  

#### 输入处理

- `handleCanvasClick()` - 处理画布点击事件

- `handleMouseMove()` - 处理鼠标移动和拖拽

- `handleWheelZoom()` - 处理缩放

- `handleKeyDown()` - 处理键盘事件

  

#### 游戏逻辑

- `enterInputMode()` - 进入输入模式

- `confirmDraft()` - 确认提交诗句

- `exitDraftMode()` - 退出编辑模式

- `undoLastPoem()` / `redoLastPoem()` - 撤销/重做

- `exportGame()` / `importGame()` - 存档导入导出
